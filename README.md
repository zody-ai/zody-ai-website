# Zody AI compliance site

Static website prepared for GitHub Pages.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this folder to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Set the source to **Deploy from a branch**.
5. Select the main branch and root folder.
6. Save, then wait for GitHub Pages to publish the site.
7. Optional: add a custom domain such as `zodyai.co.uk` in the Pages settings.

## Before going live

- Replace placeholder company number and registered office.
- Add real contacts for support, privacy and security.
- Review `pages/privacy.html` and `pages/terms.html` with a professional adviser.
- Search the site for the word `placeholder` and remove all placeholders.

## Structure

- `index.html` homepage
- `pages/` deep-dive compliance pages and policy pages
- `assets/css/styles.css` shared styling
- `assets/js/site.js` small helper for nav highlighting and year
