Compliance areas

HMRC’s terms of use for production access group their questions into organisation details, marketing, development practices, service management, handling personal data, software security, fraud prevention data and customer authorisation. This site mirrors that structure to make your answers easy to follow.

How to use this site

This site serves as the public-facing trust layer for bank account applications, payment provider onboarding, accounting integrations and regulatory due diligence. It complements your product marketing site rather than replacing it.

Publishing checklist

  • Push this repository to GitHub and enable GitHub Pages from the root directory.
  • Point zody-ai.uk to the GitHub Pages domain via your DNS provider.
  • Have a solicitor review the privacy policy and terms before live commercial use.

Why this structure works

Fast to publish

GitHub Pages can publish static HTML, CSS and JavaScript straight from a repository, with optional custom domains for a more credible public presence.

Aligned to HMRC review questions

The page hierarchy directly follows HMRC’s production access review topics, which helps reviewers find the exact answer they are looking for without digging.
